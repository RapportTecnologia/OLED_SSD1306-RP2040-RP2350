__IOM uint32_t RESETS_Type::RESET

Reset control. If a bit is set it means the peripheral is in reset. 0 means the peripheral's reset is deasserted.

RESET_DONE

If a bit is set then a reset done signal has been returned by the peripheral. This indicates that the peripheral's registers are ready to be accessed.
