__IOM uint32_t USB_DPRAM_Type::EP0_IN_BUFFER_CONTROL

Buffer control for both buffers of an endpoint.

Fields ending in a _1 are for buffer 1. Fields ending in a _0 are for buffer 0. Buffer 1 controls are only valid if the endpoint is in double buffered mode.
