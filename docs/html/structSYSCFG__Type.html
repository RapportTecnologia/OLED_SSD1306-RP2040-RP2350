◆ PROC_IN_SYNC_BYPASS

__IOM uint32_t SYSCFG_Type::PROC_IN_SYNC_BYPASS

For each bit, if 1, bypass the input synchronizer between that GPIO and the GPIO input register in the SIO.

The input synchronizers should generally be unbypassed, to avoid injecting metastabilities into processors. If you're feeling brave, you can bypass to save two cycles of input latency. This register applies to GPIO 0...29.

__IOM uint32_t SYSCFG_Type::PROC_IN_SYNC_BYPASS_HI
