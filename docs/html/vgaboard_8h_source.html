18// Audio pins. I2S BCK, LRCK are on the same pins as PWM L/R.
19// - When outputting I2S, PWM sees BCK and LRCK, which should sound silent as
20// they are constant duty cycle, and above the filter cutoff
21// - When outputting PWM, I2S DIN should be low, so I2S should remain silent.
