◆ pico_board_cmake_set()

pico_board_cmake_set ( PICO_PLATFORM  ,
rp2040   
)

Mask of pins that are used by the shield and not used by the board, eq. 0x3fc0033f.

divisor for SPI clock (default 4), must be divisible by 2

This is critical for some flash chips, and can be set to 4 for some, but needs to be at least 32 for others. Set using -DCLKDIV=<N*2> in cmake.

Nota
Increasing this value will help with BOOTLOOPs, but will slow down the flash chip.
