87// PICO_CONFIG: XOSC_HZ, Crystal oscillator frequency in Hz, type=int, default=12000000, advanced=true, group=hardware_base
88// NOTE: The system and USB clocks are generated from the frequency using two PLLs.
89// If you override this define, or SYS_CLK_HZ/USB_CLK_HZ below, you will *also* need to add your own adjusted PLL set-up defines to
90// override the defaults which live in src/rp2_common/hardware_clocks/include/hardware/clocks.h
91// Please see the comments there about calculating the new PLL setting values.
